Les éléments EniBook sont des éléments HTML personnalisés (custom element) représentés par des balises du type <element-name-it></element-name-it> dont le nom se termine toujours par le suffixe -it.

Les développements sont réalisés en typescript et sont accessibles depuis l’API (Application Programming Interface).
